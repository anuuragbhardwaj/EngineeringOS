# Knowledge CLI

| Command | Description |
|---------|-------------|
| `knowledge status` | Platform summary |
| `knowledge search <query>` | Text search |
| `knowledge explain <id>` | Full traceability |
| `knowledge graph <id>` | Relationship graph |
| `knowledge validate <id>` | Validation report |
| `knowledge promote <id>` | Promote scope |
| `knowledge history` | Promotion audit |
| `knowledge stats` | Aggregates |
| `knowledge export <file>` | JSON bundle |
| `knowledge import <file>` | Import bundle |

Options: `--scope`, `--type`, `--json`, `--to`, `--reason`, `--depth`
