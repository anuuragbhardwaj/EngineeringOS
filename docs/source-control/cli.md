# Source Control CLI

| Command | Description |
|---------|-------------|
| `repo status` | Repository status |
| `repo validate` | Health validation |
| `repo commit` | Create commit (EM approval) |
| `repo push` | Push (EM approval) |
| `repo history` / `repo log` | Commit history |
| `repo branches` | List branches |
| `repo checkout` | Switch branch |
| `repo tag` | Create tag |
| `repo release` | Prepare release plan |
| `repo diff` | Show diff |
| `repo stage` / `repo unstage` | Staging |
| `repo doctor` | Diagnostics |
| `repo approve` | EM approval |
