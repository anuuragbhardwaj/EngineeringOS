# Autonomous CLI

| Command | Description |
|---------|-------------|
| `work <goal>` | Start goal-based execution |
| `continue` | Continue autonomous execution |
| `stop` | Stop runner |
| `goals` | List goals |
| `blockers` | Active blockers |
| `approvals approve <gate>` | EM approval |
| `autonomy status` | Full status |
| `supervise` | One supervision cycle |
| `monitor` | Monitor snapshot |
| `heartbeat` | Runner heartbeat |
| `recover` | Session recovery |
| `explain` | Explain state |
| `decisions` | Decision history |
